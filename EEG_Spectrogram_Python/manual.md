# 🧠 EEG スペクトログラム生成プロセスまとめ

---

## 1. 🎯 目的

脳波（EEG）信号の時間変化に伴う **周波数成分（スペクトル強度）** を可視化するために、  
**離散フーリエ変換（DFT）** を用いてスペクトログラムを作成する。

---

## 2. 📘 入力データ

**ファイル名:** `signal_data.csv`  
**内容:**

| Time [sec] | Signal Raw [uV] |
| ----------- | ---------------- |
| 0.000       | -5.8996          |
| 0.001       | 7.0576           |
| 0.002       | 33.1945          |
| ...         | ...              |

- サンプリング周波数: $f_s = 1000 \, \mathrm{Hz}$

---

## 3. ⚙️ 前処理

信号を一定長の時間窓に分割して解析する。

### 3.1 窓関数

ハミング窓（Hamming window）を使用：

$$
w[n] = 0.54 - 0.46 \cos\left( \frac{2\pi n}{N-1} \right), \quad 0 \le n < N
$$

### 3.2 分割パラメータ

| パラメータ | 意味 | 例 |
| ----------- | ---- | --- |
| `nperseg`   | 1フレームの長さ [samples] | 1024 |
| `noverlap`  | 隣接フレームの重複長 [samples] | 512 |
| `hop`       | フレーム移動量 = `nperseg - noverlap` | 512 |
| `window`    | 使用する窓関数 | Hamming |

---

## 4. 🔍 フーリエ変換（DFT）

各時間フレームごとに、次の式で **離散フーリエ変換（DFT）** を計算する：

$$
X(k, m) = \sum_{n=0}^{N-1} x[n + mH] \, w[n] \, e^{-j \frac{2\pi kn}{N}}
$$

- $k$: 周波数ビン（0〜N/2）  
- $m$: 時間フレーム番号  
- $H = \text{hop}$: フレーム移動幅  

この処理を全時間フレームに対して繰り返すことで、時間と周波数の両方に依存するスペクトルを得る。

---

## 5. ⚡ パワースペクトルの算出

各周波数ビンにおける強度（パワー）を求める：

$$
P(k, m) = \frac{1}{f_s \, \sum_{n=0}^{N-1} w[n]^2} \, |X(k, m)|^2
$$

> 窓関数の二乗和とサンプリング周波数で正規化することで、  
> エネルギー密度として解釈できるようにする。

---

## 6. 📏 対数スケーリング（dB変換）

対数スケールに変換して人間の知覚特性に近づける：

$$
P_{\mathrm{dB}}(k, m) = 10 \log_{10} \left( P(k, m) + 10^{-12} \right)
$$

- $10^{-12}$ は数値安定化項（0除算防止）

---

## 7. 🖼️ スペクトログラムの描画

### 7.1 軸定義

| 横軸 | 秒 [s] | 時間フレーム中心位置 |
| 縦軸 | ヘルツ [Hz] | 周波数ビン $(k f_s / N)$ |
